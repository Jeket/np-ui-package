Sidepanel
Module to import
HTML
Property Description Type Default
top Set top position in px. Set this '0', if it is top side panel string
bottom Set bottom position in px. Set this '0', if it is bottom side panel string
left Set left position in px. Set this '0', if it is left side panel string
right Set right position in px. Set this '0', if it is right side panel string
height Set height, if pass as number then it will take it in px. Like '100%', '100px', 200 string | number
width Set width, if pass as number then it will take it in px. Like '100%', '100px', 200 string | number
showCloseIcon Show close icon boolean true
closeOnClickOutside Close panel on click of outside area boolean true
hasBackDrop Show backdrop or not boolean true
backDropClass Set backdrop class string np-sp-backdrop
styleClass Set extra class on component string
inputId Set id attribute string
Api Description Parameters
open() Open side panel
close() Close side panel
