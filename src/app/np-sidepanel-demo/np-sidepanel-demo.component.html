Module to import
HTML
Property Description Type Default
top Set top offset. Set this '0', if it is top side panel. Values should be like '100%', '100px', 200. If pass as number then it will take it in px. string
bottom Set bottom offset. Set this '0', if it is bottom side panel. Values should be like '100%', '100px', 200. If pass as number then it will take it in px. 200. string
left Set left offset. Set this '0', if it is left side panel. Values should be like '100%', '100px', 200. If pass as number then it will take it in px. string
right Set right offset. Set this '0', if it is right side panel. Values should be like '100%', '100px', 200. If pass as number then it will take it in px. string
height Set height. Values should be like '100%', '100px', 200. If pass as number then it will take it in px. string | number
width Set width. Values should be like '100%', '100px', 200. If pass as number then it will take it in px. string | number
showCloseIcon Show close icon boolean true
closeOnClickOutside Close panel on click of outside area boolean true
hasBackDrop Show backdrop or not boolean true
backDropClass Set backdrop class string np-sidepanel-backdrop
styleClass Set extra class on component string
inputId Set id attribute string
Here only 1 position offset can be set on both side. Another one will be unset automatically. Also needs to add height and width as per offset.
When setting top or bottom sidepanels then set width as '100%', and for left or right sidepanels set height as '100%'.
Method Description Parameters
onOpen() Callback on open
onClose() Callback on close
Api Description Parameters
open() Open side panel
close() Close side panel
