Others

key

To set key column in data grid, set key property with column's dataField value in grid defination.

Example: [key]="'Id'"

height

To set height of data grid, set height property in grid defination.

Example: [height]="500". No need to pass "px" as this value will be considered in pixels only.

width

To set width of data grid, set width property in grid defination.

Example: [width]="1000". No need to pass "px" as this value will be considered in pixels only.

title

To show title message on grid, set title property in grid defination.

Example [title]="'All Employees in company.'"

Default message is "No Data Found."

noDataMessage

To show message when no records found, set noDataMessage property in grid defination.

Example [noDataMessage]="'No Employee found.'"

pageSize

To set default pagesize for data grid, set pageSize property.
