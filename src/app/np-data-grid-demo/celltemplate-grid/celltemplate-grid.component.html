Cell Template Example

To add cell template in data grid add <ng-template> in html file and get that template as TemplateRef<any> in component.

Now set that template reference in cellTemplate property in column definations.

In HTML file:

<!-- Template for active column -->
<ng-template #activeColumnTemplate let-row="row">
    <input type="checkbox" [checked]="row.Active" disabled>
</ng-template>

In Component file:

@ViewChild("activeColumnTemplate", {{'{'}} static: true {{'}'}}) activeColumnTemplate: TemplateRef<any>;

{{row.BirthDate | date:'dd/MM/yyyy'}}
