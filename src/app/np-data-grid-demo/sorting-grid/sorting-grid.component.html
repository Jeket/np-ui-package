Sorting example

To add sorting in data grid set sortEnable to "true" in column definations.
Default value for sortEnable is "false".

To set multicolumn sorting, set sortEnable to "true" and multiColumnSortEnable to "true" in grid defination.
Default value for multiColumnSortEnable is "false".

To set Multi column sort enable for both:
Client side sorting

In Client side data grid, all sorting oprations will be done on client side.

Server side sorting

In Server side data grid, sortColumns will be passed to onLoadData(options:LoadOptions).
Which will contains array of all sorting columns as {{ '{' }} column : string, sortDirection: string {{'}'}}

Possible values for sortDirection are asc, desc

Options: {{currentLoadOptions | json}}
