Sorting Example.

To add sorting in data grid set sortEnable to "true" in column definations.

To set multicolumn sorting, set multiColumnSortEnable to "true" in grid defination.

Multi column sort enable:

Default value for multiColumnSortEnable is "false".

Client side sorting:

In Client side data grid all sorting oprations will be done on client side.

Server side sorting:

In Server side data grid sortColumns will be passed to onLoadData(options:LoadOptions). Which will contains array of all sorting columns as {{ '{' }} column : string, sortDirection: string {{'}'}}

Possible values for sortDirection are asc, desc.

options:{{currentLoadOptions | json}}
