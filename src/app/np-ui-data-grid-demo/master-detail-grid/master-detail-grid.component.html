Master-Detail Example

To set Master Detail tables in data grid, create <ng-template> in HTML file and set it in masterDetailTemplate in grid defination.

singleRowExpand

To set single row expand mode set singleRowExpand to "true" in grid defination.

expandRowOnClick

To expand/collaps details row on row click set expandRowOnClick to "true" in grid definations.
