How to add np-ui-lib package to your application?

Step 1: Install node package using below command. 
$ npm install np-ui-lib

Step 2: Add CSS file to your application's style.css. 
@import "~@angular/cdk/overlay-prebuilt.css";
@import "np-ui-lib/style.css";

Below packages already needs to be pre-installed

"@angular/common": "^10.0.0",
"@angular/core": "^10.0.0",
"@angular/cdk": "^10.0.0",
"@angular/forms": "^10.0.0",
"@angular/router": "^10.0.0",
"rxjs": "^6.5.5"
