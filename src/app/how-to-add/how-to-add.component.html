NP-UI-LIB

Native Angular UI Components and Design Framework

Current Version: 10.3.3, Angular Version: 10.0.0+

     
Why np-ui-lib?
  • An enterprise level UI design
  • 34+ components
  • Free and open source
  • Fast and easy to implement
Best Practices
  • Followed best coding practices
  • Responsive UI design
  • Good documenation
  • Flexible to customization
How to add np-ui-lib package to your application?

Step 1: Install node package using below command. 
$ npm install np-ui-lib

Step 2: Add CSS file to your application's style.css. 
@import "~@angular/cdk/overlay-prebuilt.css";
@import "np-ui-lib/style.css";
Below packages already needs to be pre-installed
 
"@angular/common": "^10.0.0",
"@angular/core": "^10.0.0",
"@angular/cdk": "^10.0.0",
"@angular/forms": "^10.0.0",
"@angular/router": "^10.0.0",
"rxjs": "^6.5.5"
