NP-UI-LIB

Native Angular UI Components and Design Framework

Current Version: 11.0.0, Angular Version: 11.0.0+

NPM Github Documentation

Why np-ui-lib?
  • An enterprise level UI design
  • 41+ components
  • Free and open source
  • Fast and easy to implement
  • Theming and i18N support

Best Practices
  • Followed best coding practices
  • Responsive UI design
  • Good documentation
  • Flexible to customization
  • WAI-ARIA included

How to add np-ui-lib package to angular application?
Step 1: Install node package using below command 
$ npm install np-ui-lib
Step 2: Import below style sheets to application's styles.scss file 
@import "~@angular/cdk/overlay-prebuilt.css";
@import "np-ui-lib/styles/default-theme.scss";

Below packages already needs to be pre-installed

 
"@angular/common": "^11.0.6",
"@angular/core": "^11.0.6",
"@angular/cdk": "^11.0.3",
"@angular/forms": "^11.0.6",
"@angular/router": "^11.0.6",
"rxjs": "^6.6.3"
