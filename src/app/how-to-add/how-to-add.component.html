Welcome to np-ui-lib angular components

This package supports Angular version 9+.

How to add np-ui-lib package to your application?

Step 1: Add node package using below command.

$ npm install np-ui-lib

Step 2: Add CSS file to your application's style.css.

@import "np-ui-lib/style.css";

Step 3: Import component module from np-ui-lib to application module.

Step 4: Enjoy and save time with adding components easily.
