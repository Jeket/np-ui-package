<np-tabs>
Property Description Type Default
verticalTabs Set vertical tabs boolean false
height Set height of tabs in px number
styleClass Set extra class on component string
inputId Set id attribute string
<np-tab>
Property Description Type Default
title Set title for tab string | TemplateRef<any>
active Get/Set flag for current active tab, only one tab will be active at a time boolean
disabled Set tab as disabled boolean false
iconCss Set CSS for icon like 'fa fa-home' string
isLoadFirstTime Internal property, will be true on loading tab first time, after that will be set to false
Helpful while making lazy loading tabs, and fetching data on click of tab 		string
inputId Set id attribute string
<np-tabs>
Apis Description Parameters
selectTabByIndex(idx: number) Open tab by index, start with 0 idx: number
selectTabById(id: string) Open tab by inputId id: string
getSelectedTabIndex() Get selected tab index
getSelectedTab() Get selected tab
<np-tabs>
Methods Description Parameters
onTabChange(tab: NpTabComponent) Callback event on change of tab
